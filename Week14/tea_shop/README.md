# 茶飲專賣 - 高端茶飲網站

一個優雅而精緻的茶飲專賣網站，展示高端茶飲選擇、季節限定商品，以及靈感來自在地文化的店卡概念設計。

## 專案概述

茶飲專賣是一個響應式網站應用程式，旨在展示高級茶飲商品，具有現代、高級的美學風格。網站採用深藍綠與金色配色方案、精緻的排版和充足的留白，打造奢華的品牌體驗。

## 功能特色

### 四個主要頁面

1. **index.html** - 首頁
   - 品牌英雄區塊和引人入勝的標語
   - 快速導航卡片，連結至菜單、茶飲介紹和店卡頁面
   - 關於我們和聯絡資訊在頁尾
   - 優雅的行動號召按鈕

2. **menu.html** - 菜單頁面
   - 基於地點的菜單過濾功能（4 個分類）
   - 台北及離島、其他地區、百貨商場、服務據點
   - 左側視覺區域搭配右側菜單列表
   - 展示中杯和大杯價格
   - 下載菜單按鈕

3. **drinks.html** - 茶飲介紹頁面
   - 可過濾的茶飲選擇（經典茶款 / 季節限定）
   - 8 款經典茶飲及其描述
   - 8 款季節限定茶飲及詳細風味介紹
   - 響應式網格佈局（桌面 4 列，在行動裝置上自動調整）
   - 卡片式呈現配搭懸停效果

4. **card.html** - 店卡概念頁面
   - 店卡設計展示（6 個設計概念）
   - 各地點的文化主題描述
   - 左右並排的店舖資訊展示
   - 六個地點，各具獨特文化靈感：
     - 台北信義：現代傳承
     - 新竹濱海：海濱魅力
     - 宜蘭山林：大自然之美
     - 台中寺廟：宗教文化
     - 台南古城：歷史韻味
     - 高雄港灣：都市活力

## 色彩配置

- **深藍綠（主色）**: `#1a4d5c`
- **金色（強調色）**: `#c9a876`
- **淺色文字**: `#f0f0f0`（用於深色背景）
- **深色文字**: `#333333`（用於淺色區域）

## 排版設計

- **字型**: Georgia, 'Times New Roman', 襯線字體
- **標題**: 常規粗細，大寫樣式，寬字間距
- **本文**: 襯線字體，呈現精緻外觀

## 響應式設計

網站完全響應式，支援以下斷點：
- **桌面**: 4 列佈局，完整導航列
- **平板** (最大寬度 1024px): 3 列佈局
- **手機** (最大寬度 768px): 2 列卡片佈局，表單單列顯示
- **小型手機** (最大寬度 480px): 全寬單列佈局

## 檔案結構

```
tea_shop/
├── index.html           # 首頁
├── menu.html            # 菜單頁面（含地點過濾）
├── drinks.html          # 茶飲介紹頁面（含分類過濾）
├── card.html            # 店卡概念頁面
├── README.md            # 本說明文件
├── css/
│   └── all.css          # 合併所有樣式（全局 + 頁面專用）
├── js/
│   └── main.js          # 全局 JavaScript 功能
└── images/              # 圖片資源目錄
```

**注意**：CSS 已合併至單一檔案 `css/all.css`，包含所有全局樣式、頁面專用樣式及響應式設計。

## 導航列

所有頁面都具有一致的導航列，包含以下連結：
- 菜單
- 茶飲介紹
- 店卡
- 語言切換（EN/中文）

所有頁面都提供固定浮動菜單按鈕，方便快速訪問菜單。

## 互動功能

### 菜單頁面
- 點擊分類按鈕可按地點過濾菜單
- 不同地點菜單之間的流暢過渡
- 無需重新載入頁面即可動態更新內容

### 茶飲頁面
- 在經典茶款和季節限定之間切換
- 卡片懸停效果和視覺反饋
- 流暢的淡入淡出動畫

### 全局功能
- 根據目前頁面動態標記活躍導航連結
- 浮動菜單按鈕的懸停效果
- 響應式側邊欄和行動選單支援
- 整個網站的流暢過渡和動畫

## 瀏覽器相容性

- 支援 CSS Grid 和 Flexbox 的現代瀏覽器
- Chrome、Firefox、Safari、Edge（最新版本）
- 行動瀏覽器（iOS Safari、Chrome Mobile）

## 快速開始

1. 複製或下載專案檔案
2. 在網頁瀏覽器中開啟 `index.html`
3. 使用導航選單在四個主要頁面之間導覽

## 自訂設定

### 色彩
在 `css/style.css` 中編輯 CSS 變數：
```css
:root {
  --primary-dark: #1a4d5c;
  --accent-gold: #c9a876;
  --text-light: #f0f0f0;
  --text-dark: #333333;
}
```

### 菜單資料
在 `menu.html` 中編輯 `menuData` 物件來更新飲品資訊。

### 茶飲資料
在 `drinks.html` 中編輯 `drinksData` 物件來更新茶飲描述。

## 專案結構詳解

### CSS 組織

**all.css** - 包含：
- 全局樣式和變數
- 標題和導航列樣式
- 頁尾樣式
- 通用元素（按鈕、區段、容器）

- 事件委派設定

- menu.html 的菜單過濾
- drinks.html 的茶飲分類切換
